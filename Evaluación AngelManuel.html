1: ¿Que es una Variable?

Una variable en informática es un espacio de memoria identificado por un nombre, que se utiliza para almacenar y manipular datos durante la ejecución de un programa. Su valor puede cambiar a lo largo del tiempo, lo que permite que el programa sea dinámico y flexible.

2: ¿Que es una constante?

Una constante es similar a una variable en cuanto a que almacena un valor en memoria, pero con una diferencia clave: su valor no puede modificarse una vez definido. Es decir, permanece inmutable durante toda la ejecución del programa.

3: Tipos de datos:

Un tipo de dato en informática es una clasificación que se le da a la información que puede manejar un programa. Define qué clase de valor puede tener una variable y qué operaciones se pueden hacer con ese valor. Por ejemplo, algunos tipos de datos comunes son números, texto y valores de verdadero o falso.

4: ¿Que son los datos estructurados y los no estructurados?

Los datos estructurados son aquellos que están organizados de forma precisa y predecible. Normalmente se almacenan en tablas, bases de datos o archivos con un formato fijo, como hojas de cálculo. Cada dato tiene un lugar definido, como una columna para nombres, otra para fechas, otra para números. Esto permite que los programas los lean, busquen y analicen fácilmente. Por ejemplo, una lista de clientes con nombre, edad y dirección es un conjunto de datos estructurados. Todo está ordenado y etiquetado. En cambio, los datos no estructurados no siguen un formato específico. Son más libres y variados. Pueden ser textos largos, imágenes, audios, videos o correos electrónicos. Aunque contienen mucha información útil, no están organizados de forma que una máquina pueda entenderlos fácilmente sin ayuda. Para analizarlos se necesitan herramientas más avanzadas, como inteligencia artificial o programas de reconocimiento. Un ejemplo de datos no estructurados sería una conversación por chat, una foto en redes sociales o una grabación de voz. No hay columnas ni etiquetas claras, pero aún así pueden contener datos valiosos

5: Diferencia entre dato e informacion:

En informática, un dato es una unidad mínima de información que por sí sola no tiene un significado completo. Puede tratarse de un número, una palabra, una fecha o cualquier otro valor que represente un hecho simple. Los datos son elementos crudos, sin contexto, que necesitan ser organizados o interpretados para adquirir sentido. Por ejemplo, el número 85, la palabra “rojo” o la fecha “10 de abril” son datos, pero no nos dicen mucho si no sabemos a qué se refieren. Por otro lado, la información es el resultado de procesar, organizar o interpretar los datos de manera que adquieran significado y puedan ser útiles para tomar decisiones, resolver problemas o comunicar algo. Cuando los datos se combinan con un contexto adecuado, se transforman en información. Por ejemplo, si tomamos el dato “85” y lo colocamos en la frase “La temperatura actual es de 85 grados”, entonces ese dato se convierte en información comprensible y útil. La diferencia entre ambos conceptos radica en el nivel de procesamiento y utilidad. Mientras que los datos son materia prima, la información es el producto final que se obtiene al analizarlos y darles un propósito. En los sistemas informáticos, este proceso de transformación es fundamental para que los usuarios puedan interpretar los datos y tomar decisiones basadas en ellos.

6: ¿Que es una API? y ejemplo:

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas que permite que dos programas se comuniquen entre sí. Es como un puente que conecta diferentes sistemas para que puedan intercambiar datos o funciones sin necesidad de conocer cómo están construidos internamente. Por ejemplo, cuando una aplicación de clima muestra el pronóstico, no calcula el clima por sí sola. En lugar de eso, utiliza una API para pedirle esa información a un servicio meteorológico. La API recibe la solicitud, busca los datos y los devuelve a la aplicación para que los muestre.

7: ¿Que son los manejadores de bases de datos?

Un Sistema Manejador de Bases de Datos (SMBD) es un conjunto de programas que actúan como intermediarios entre los usuarios y las bases de datos. Su función principal es facilitar el almacenamiento, modificación, consulta y administración de grandes volúmenes de datos de manera estructurada y coherente.

8: ¿Que son los contenedores Docker?

Docker es una plataforma de código abierto que automatiza el despliegue de aplicaciones dentro de contenedores. Un contenedor es una unidad estándar de software que empaqueta el código de una aplicación junto con todas sus dependencias (librerías, herramientas del sistema, configuraciones) para que funcione de manera uniforme en cualquier entorno.

9: ¿Como ejecutar un contenedor en docker? y ejemplo:

Ejecutar un contenedor en Docker es un proceso directo que te permite poner en marcha una aplicación en segundos:

Paso 1:

Instala Docker en tu sistema Descarga e instala Docker desde su sitio oficial, según tu sistema operativo (Windows, macOS o Linux).

Paso 2:

Busca una imagen de contenedor Entra a Docker Hub (el repositorio oficial de imágenes) y elige una imagen que quieras usar, como una de Nginx, Python, Node.js, PostgreSQL, etc.

Paso 3:

Descarga la imagen seleccionada Usa Docker para obtener esa imagen y tenerla lista en tu máquina.

Paso 4:

Crea y ejecuta un contenedor Lanza un contenedor a partir de esa imagen. Puedes asignarle un nombre, definir qué puertos usar y si quieres que se ejecute en segundo plano.

Paso 5:

Verifica que el contenedor esté funcionando Consulta la lista de contenedores activos para asegurarte de que el tuyo está corriendo correctamente.

Paso 6:

Accede a la aplicación o servicio Abre tu navegador o herramienta de conexión (como un cliente web o de base de datos) y conéctate al puerto que configuraste.

Paso 7:

Detén el contenedor cuando termines Puedes pausar o detener el contenedor para liberar recursos.

Paso 8:

Elimina el contenedor si ya no lo necesitas Esto borra el contenedor de tu sistema, aunque la imagen base seguirá disponible para crear nuevos.

10: Terminos Basicos: www,ip, dns, isp, vpn y https

Estos términos son fundamentales para entender cómo funciona Internet y cómo se conectan los dispositivos entre sí.

• www (World Wide Web):

Es el sistema de páginas web interconectadas que se acceden a través de navegadores. No es lo mismo que Internet, sino una parte de él que permite visualizar contenido como texto, imágenes y videos.

• IP (Internet Protocol):

Es un número único que identifica a cada dispositivo conectado a una red. Puede ser pública (visible en Internet) o privada (dentro de una red local). Ejemplo: 192.168.1.1.

• DNS (Domain Name System):

Es el sistema que traduce nombres de dominio (como google.com) en direcciones IP. Así, puedes escribir un nombre fácil de recordar en lugar de una serie de números.

• ISP (Internet Service Provider):

Es la empresa que te proporciona acceso a Internet. Ejemplos comunes son Movistar, Cantv, Fibex, etc.

• VPN (Virtual Private Network):

Es una red privada virtual que cifra tu conexión y oculta tu dirección IP. Se usa para proteger tu privacidad, acceder a contenido restringido o conectarse de forma segura a redes corporativas.

• HTTPS (HyperText Transfer Protocol Secure):

Es la versión segura del protocolo HTTP. Usa cifrado para proteger la información que se transmite entre tu navegador y el sitio web. Se reconoce por el candado en la barra de direcciones.
